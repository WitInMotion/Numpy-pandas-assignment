# NumPy and Pandas Assignment

This project contains a data analysis assignment completed using **NumPy** and **Pandas** in **Google Colab**.

## Project Overview
The assignment demonstrates:
- Simulation of sales data using NumPy
- Financial analysis with array operations
- Data loading, cleaning, and analysis using Pandas
- Grouping, aggregation, and data transformation
- Business-focused insights based on real datasets

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- Google Colab

## How to Run
1. Open the notebook in Google Colab or Jupyter Notebook
2. Run all cells from top to bottom
3. Ensure the CSV file is in the same directory as the notebook
